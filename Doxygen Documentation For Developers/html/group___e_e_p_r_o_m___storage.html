Integrated BMS Firmware V2_3 Sodium & Lithium ION BMS Include Adafruit INA219 current/voltage sensor library Include hardware serial library for interrupt handling Include STM32 timer interrupt library for periodic tasks Include STM32 hardware abstraction layer Include EEPROM library for persistent storage
